Extensão Desktop

Extensão
"Levante-se!"

Pausas ativas para pessoas com hiperfoco, TDAH e profissionais que esquecem de se movimentar durante o trabalho.

Ver Demonstração Sobre o Projeto
Demonstração
Chrome Edge Manifest V3
01

Desafio

Pessoas com hiperfoco — como autistas e pessoas com TDAH — e profissionais que trabalham longas horas em frente ao computador frequentemente esquecem de fazer pausas para se levantar, alongar e hidratar.

O cenário atual revela que:

  • ✦ Hiperfoco: Pessoas neurodivergentes podem ficar imersas em tarefas por horas sem perceber a passagem do tempo.
  • ✦ Trabalho remoto: Com o home office, a linha entre trabalho e descanso se tornou mais tênue, aumentando o sedentarismo.
  • ✦ Dados: Estudos mostram que pausas regulares a cada 25-30 minutos melhoram a produtividade e reduzem dores musculares.
  • ✦ Oportunidade: Uma extensão de navegador simples, não intrusiva e personalizável pode ser a solução ideal.

05

Solução

Como funciona a extensão

Uma extensão para navegadores Chrome/Edge que interrompe o hiperfoco através de um popup com mecanismo de "forçamento" — o usuário só consegue fechá-lo após 20 cliques ou 3 minutos de espera.

🖱️

20 Cliques

O botão "Fechar" exige 20 cliques para ser liberado. Um contador mostra o progresso (ex: 5/20, 12/20), incentivando o usuário a completar a ação.

⏳

3 Minutos de Espera

Alternativa aos cliques: um timer de 3 minutos que, ao final, libera o fechamento automático do popup — dando tempo para o usuário se desconectar do hiperfoco.

⚡ Funcionalidades Principais

⏰

Timer configurável

5, 15, 30, 60 minutos ou valor personalizado

🖱️

Sistema de 20 cliques

Botão com contador progressivo (5/20, 12/20)

⏳

Espera de 3 minutos

Timer alternativo para fechamento automático

💧

Lembrete de hidratação

A cada 2 ciclos, lembrete para beber água

🔄 Como funciona o mecanismo de "forçamento"

1

Timer dispara

O popup aparece sobrepondo a tela com fundo escuro

2

Escolha a ação

Clique 20 vezes no botão ou aguarde 3 minutos

3

Pausa realizada

Popup fecha e a dica de alongamento é exibida

Progresso dos cliques 12 / 20

Faltam 8 cliques para fechar o popup

Requisitos do

Funcionais · Não Funcionais

⚙️ Requisitos Funcionais

O que o sistema deve fazer

RF01

Timer configurável

O usuário deve poder definir o intervalo das pausas: 5, 15, 30, 60 minutos ou valor personalizado.

RF02

Popup de interrupção

Ao final do timer, um popup deve aparecer sobrepondo a tela com fundo escuro, exibindo uma dica de alongamento ou hidratação.

RF03

Sistema de 20 cliques

O popup deve ter um botão "Fechar" que exige 20 cliques para ser liberado. Um contador deve mostrar o progresso (ex: 5/20, 12/20).

RF04

Sistema de espera de 3 minutos

O popup deve ter um timer de 3 minutos que, ao final, libera o fechamento automaticamente como alternativa aos cliques.

RF05

Personalização do "forçamento"

O usuário deve poder ajustar a quantidade de cliques necessários (ex: 10, 15, 20, 30) e o tempo de espera (ex: 2, 3, 5 minutos).

🔧 Requisitos Não Funcionais

Como o sistema deve se comportar

RNF01

Performance

A extensão deve consumir menos de 30MB de memória em idle e menos de 50MB durante a exibição do popup.

RNF02

Disponibilidade

A extensão deve funcionar offline, armazenando todas as configurações e estatísticas localmente (chrome.storage.local).

RNF03

Compatibilidade

A extensão deve ser compatível com Chrome 88+, Edge 88+ e demais navegadores baseados em Chromium.

RNF04

Privacidade

Nenhum dado do usuário deve ser enviado para servidores externos. Todas as informações ficam armazenadas localmente.
